# Book Database API

REST API для управления книгами, жанрами и участниками (авторы, редакторы, иллюстраторы).

## Технологии

- Python 3.10+
- FastAPI
- SQLAlchemy 2.0
- PostgreSQL 13+
- Pydantic
- Docker & Docker Compose
